# Your knowledge base is being set up

This site is generated from your **sources** and **ontology** using Cursor. Until the pipeline runs, there's nothing to show yet — and that's expected.

<Callout type="info" title="What to do next">
  Open your project in **Cursor** and follow the **README** in the root of your
  repo. It walks you through creating your ontology, ingesting content, and
  publishing docs to this site.
</Callout>

## Where am I in the pipeline?

Not sure what state your project is in? See the [Setup guide](/docs/guides/setup-guide) for a walkthrough of each stage and what to do when you're stuck.

## Already have content?

If you've run the pipeline and still see this page, the `update-docs` skill may not have replaced this file yet. Open Cursor and ask it to run `update-docs` — this landing page will be replaced with your knowledge base index.
